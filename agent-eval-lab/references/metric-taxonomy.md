# Metric Taxonomy

## L1 Mandatory Metrics

- Output parseability
- Output format compliance
- Required-field completeness
- Tool-call success or final-answer presence

## L2 Capability Metrics

- Classification accuracy
- Precision and recall
- Extraction exact or partial match
- Numeric extraction error
- Retrieval coverage
- Auditable reasoning-step validity
- Instruction-following pass rate

## L3 Business Metrics

- Forbidden-word cleanliness
- Brand or tone compliance
- Business-rule hit rate
- Domain safety compliance
- Agent-specific workflow completion
